Navigation Balise "a" Attribut "href" et "target"

  • WebsiteAttribut target="_blank" pour ouvrir dans une nouvelle fenêtre ou un nouvel onglet.
  • Websiteattribut target="_parent" pour ouvrir dans le contexte de navigation parent à celui en cours. Si aucun parent n’existe, alors ça revient à écrire "_self".
  • WebsiteAttribut target="_self" pour ouvrir dans la fenêtre courante sur le même onglet (comportement par défaut), donc autant économiser des lignes de code.
  • WebsiteAttribut target="_top" pour ouvrir dans le contexte de navigation le plus haut (parent, grand-parent, etc.). S’il n’y a pas d’ancêtre, alors ça revient à écrire "_self".
